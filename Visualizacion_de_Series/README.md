# Proyecto Visualización de Series

## Descripción  
Este proyecto trata de mostrarle al cliente series interesante para maratonear cargando las series desde una API externa. Esto acompañado de una paleta de colores agradable y un diseño intuitivo para mejorar la experiencia del usuuario.

## Estructura del proyecto  
Aquí un resumen de las carpetas y archivos más importantes (según lo que posee tu proyecto):  
- `index.html` — punto de entrada principal de la aplicación frontend(JAMstack app).  
- `.css` — estilos para la aplicación.  
- `.js` — código JavaScript para manejar la lógica de visualización.  
 

## Funcionalidades  
Algunas de las cosas que se pueden hacer con el proyecto:  
- Ver series guardadas en el localStorage.
- Ver series por lenguaje o género.
- Diseño responsivo para que funcione en distintos tamaños de pantalla.
